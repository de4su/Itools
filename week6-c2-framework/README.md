# Week 6: Command & Control Framework

Educational C2 framework demonstrating command and control concepts.

## Features
- Client-server architecture
- Remote command execution
- Multi-client support capability
- Educational purpose implementation

## Usage

Start C2 server:
```bash
python c2_server.py
```

Connect client (in another terminal):
```bash
python c2_client.py
```

Connect to remote server:
```bash
python c2_client.py 192.168.1.100
```
